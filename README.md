## 📌 Overview

This repository contains my solutions to the **JPMorgan Chase Quantitative Research Virtual Experience Program** on Forage.

Throughout the simulation, I worked as a **Quantitative Research Analyst**, solving practical financial problems involving:

- 📈 Natural Gas Price Forecasting
- 💰 Storage Contract Pricing
- 🤖 Probability of Default (PD) Modeling
- 📊 FICO Score Quantization using Dynamic Programming

The project combines **financial modeling, machine learning, optimization, and data analysis** to solve real-world quantitative research problems.

---

## 🚀 Project Tasks

### 📈 Task 1 – Natural Gas Price Forecasting

**Objective**
- Analyze historical natural gas prices.
- Forecast future prices for one year.
- Capture both trend and seasonal patterns.

**Techniques Used**
- Time Series Analysis
- Linear Trend Modeling
- Seasonal (Sinusoidal) Modeling
- Price Forecasting

---

### 💰 Task 2 – Storage Contract Pricing

**Objective**
- Build a pricing model for natural gas storage contracts.

**Features**
- Injection & Withdrawal Scheduling
- Storage Cost Calculation
- Contract Valuation
- Cash Flow Analysis

---

### 🤖 Task 3 – Probability of Default Prediction

**Objective**
- Predict the probability of loan default using borrower information.

**Models Implemented**
- Logistic Regression
- Random Forest Classifier

**Evaluation Metrics**
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC

The model was also used to estimate **Expected Loss (EL)** assuming a fixed recovery rate.

---

### 📊 Task 4 – FICO Score Quantization

**Objective**
- Convert continuous FICO scores into discrete credit rating buckets.

**Methodology**
- Dynamic Programming
- Maximum Log-Likelihood Optimization
- Probability of Default Estimation
- Credit Rating Mapping

This approach creates an optimal rating system suitable for future credit risk models.

# ▶️ How to Run

Clone the repository:


```bash

```

Open the notebooks:

